# Type convertor for photos 

This is a simple type (png, heic, tiff) converter for photo type - jpg


## Getting Started
1. git clone "https://github.com/VladikaAndrii/Convert-photo-types.git"
2. cd "path to directory of convertor"
3. run "pip install -r requirements.txt"
4. Place the files to be changed in a folder "photo/"
5. run "python main.py"
6. Changed file will be in folder "photo/"
7. Enjoy.
